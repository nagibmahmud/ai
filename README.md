## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation

1. Clone or download the project files
2. Open `index.html` in your web browser
3. Start using the AI features!

### Running Locally

Simply open the `index.html` file in your browser:

```bash
# On Windows
start index.html

# On macOS
open index.html

# On Linux
xdg-open index.html
```

Or use a local development server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.

## Customization

### Changing Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;      /* Main accent color */
    --primary-hover: #4f46e5;      /* Hover state */
    --bg-dark: #0f0f23;            /* Background */
    --bg-sidebar: #1a1a2e;         /* Sidebar background */
    --bg-main: #16213e;            /* Main content background */
}
```

### Adding New Features

1. Add a new button in the sidebar (`index.html`)
2. Create feature-specific UI in `loadFeatureUI()` method
3. Add response templates in `generateResponse()` method
4. Style the new feature in `styles.css`
